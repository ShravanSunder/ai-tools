# Subagent Review

Codex-first review swarm skill.

The skill defines a reusable review method:

- Build a shared review packet.
- Dispatch specialist Codex reviewer subagents.
- Include `agy` as external counsel when available.
- Add Claude or Gemini only on explicit request.
- Reduce every candidate finding through evidence, failure scenario, fix, and proof.

See `SKILL.md` for the workflow and `references/` for prompts.
