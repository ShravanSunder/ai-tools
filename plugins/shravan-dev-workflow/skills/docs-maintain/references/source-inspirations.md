# Source Inspirations

These are best-practice references, not text sources to copy.

- Cursor Agent Skills `documentation-and-adrs`: document decisions, alternatives, consequences, and README roles.
- Obra `writing-plans`: exact paths, validation gates, and no placeholders for executable plans.
- Obra `verification-before-completion`: do not claim completion without current evidence.
- Shravan session memories: stale plans/specs must be reconciled against live code/docs before they drive work.
- Shravan Linear workflow: docs are truth, tickets are tracking.
- Matt Pocock `grill-with-docs`: challenge terms and decisions against existing docs and code.

Shravan adaptation:

- `AGENTS.md` is for durable agent operating rules.
- `README.md` is for humans and broad orientation.
- `docs/changelog` keeps local memory for meta-workflow changes.
- Cleanup/purge workflows must propose before destructive edits.
